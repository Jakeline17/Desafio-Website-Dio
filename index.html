Títulos e Subtítulos

Para titulos e subtitulos usamos as tags que vão de <h1> Até <h6>.Elas ajudam a estruturar o conteúdo de maneira hierárquica, sendo H1 é o titulo principal da pagina e h6 é o menos importante.

<h1> é o mais importante e deve ser usado apenas uma vez para representar o titulo principal.

<h2>Subtítulo, geralmente utilizado para seções principais da página.Importância: Menos importante que <h1>, mas ainda significativo.

<h3> é usado para subdivisões de h2 e é útil para subseções dentro de uma seção maior.

<h4>é uma subdivisão de h3 e é usado para detalhamento adicional dentro de uma subseção.

<h5>é uma subdivisão de h4 e é usado para níveis de detalhamento ainda menores.

<h6>: O nível mais baixo de cabeçalho, usado para os detalhes mais específicos.

Texto e Formatação

A seguir estão as principais tags de formatação e texto e suas respectivas funções:

<p>Define um parágrafo, organizando o texto em blocos distintos

<mark>Destaca o texto marcado, geralmente com um fundo amarelo.

<small>Exibe o texto em uma fonte menor do que a padrão.

<i> Coloca o texto em itálico, frequentemente usado para ênfase ou títulos de obras.

<u> Sublinha o texto.

<strong>Destaca o texto com importância, geralmente exibindo-o em negrito.

<hr>Insere uma linha horizontal, geralmente usada para separar conteúdos.

<sub> Exibe o texto como subscrito, útil para fórmulas químicas ou notas de rodapé.

<sup> Exibe o texto como sobrescrito, útil para expoentes em fórmulas matemáticas.

<blockquote>Define uma citação em bloco, normalmente com recuo em relação ao restante do texto.

<font>Define a fonte, tamanho e cor do texto. Seu uso é desencorajado por causa do CSS.

<del>Mostra o texto como deletado (riscado).

<abbr>Define uma abreviação ou acrônimo, podendo exibir um texto alternativo ao passar o mouse sobre ele.

Links e Ancoragem

A tag <a> é usada para criar hyperlinks que podem levar o usuário a outra página, a uma seção específica da mesma página ou até mesmo iniciar uma ação como enviar um e-mail. A estrutura básica de um hyperlink usando a tag é a seguinte:<a href="URL>"Texto do Link:</a> href: Define o destino do link. Pode ser uma URL, um caminho relativo ou uma âncora. target: Define onde abrir o link. "_blank" abre o link em uma nova aba ou janela. title: Adiciona um texto descritivo que aparece quando o usuário passa o mouse sobre o link. id: Define um identificador único para a tag, usado para criar âncoras.

Resumo:

As tags de link e ancoragem são essenciais para a navegação na web. A tag "a" permite criar links para outras páginas, seções dentro da mesma página, ou até mesmo iniciar ações como enviar um e-mail. Usar atributos como href, target, title e id permite personalizar e definir o comportamento dos links, proporcionando uma navegação eficiente e intuitiva para os usuários.

Imagens e Atributos

As tags de imagens em HTML são usadas para incorporar imagens em páginas web. A principal tag para esse propósito é a tag "IMG", que é uma tag de elemento vazio (não possui conteúdo e não requer uma tag de fechamento). A seguir, vou explicar como usa-la e seus principais atributos. A estrutura basica da tag IMG é <img src="caminho/para/imagem.jpg" alt="Descrição da imagem">
src: (Obrigatório) Especifica o caminho da imagem a ser exibida. Pode ser um caminho relativo ou uma URL. alt: (Obrigatório) Fornece um texto alternativo para a imagem, que é usado por leitores de tela e exibido se a imagem não puder ser carregada.

Atributos Comuns

src: Define o caminho da imagem. Exemplo: src="imagens/foto.jpg" alt: Fornece uma descrição textual da imagem. Exemplo: alt="Uma bela paisagem"

width e height: Especificam a largura e a altura da imagem (em pixels). Exemplo: width="300" height="200" title: Fornece um texto descritivo que aparece como uma dica (tooltip) quando o usuário passa o mouse sobre a imagem.

Listas (não) Ordenadas

Em HTML, listas ordenadas e não ordenadas são usadas para agrupar itens relacionados de uma forma estruturada. As listas ajudam a organizar o conteúdo de maneira lógica e clara para os usuários. Aqui está uma explicação detalhada sobre essas duas formas de listas:

Lista Ordenada: (<OL>)

As listas ordenadas utilizam números para organizar os itens de forma sequencial. Elas são úteis quando a ordem dos itens é importante.

Atributos Comuns

type: Define o tipo de marcador (números, letras, romanos). Exemplo: type="1" (números), type="A" (letras maiúsculas), type="a" (letras minúsculas), type="I" (números romanos maiúsculos), type="i" (números romanos minúsculos).

start: Define o número inicial da lista. Exemplo: start="3"

reversed: Exibe a lista em ordem decrescente. Exemplo: reversed

Lista Não Ordenada

As listas não ordenadas utilizam marcadores (bolinhas, círculos, quadrados) para agrupar itens. Elas são úteis quando a ordem dos itens não é importante.

Atributos Comuns

type: Define o tipo de marcador (disc, circle, square). Exemplo: type="disc" (padrão), type="circle" (círculo), type="square" (quadrado).

Resumo

Listas Ordenadas (<OL>): Utilizam números para os itens. São úteis quando a ordem importa. Atributos importantes incluem type, start e reversed. Listas Não Ordenadas (<UL>): Utilizam marcadores (bolinhas, círculos, quadrados). São úteis quando a ordem não importa. O atributo type define o tipo de marcador.
